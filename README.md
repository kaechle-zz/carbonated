# Carbonated
A boilerplate for new front-end projects using npm and gulp 4.

## Psuedocode
```
function compile scss
  compile scss in src dir in order
  pipe to distro dir

function watch
  if changes in scss
    compile scss
    refresh browser
  else
    refresh browser

function uglify css
  compile scss
  uglify css

function uglify js
  uglify js

function uglify code
  uglify css
  uglify js

function build
  compile scss
  uglify code
```
